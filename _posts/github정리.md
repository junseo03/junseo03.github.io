현재 Git 상태 확인
{% highlight ruby %}
git status
{% endhighlight %}

Git add 하기
{% highlight ruby %}
 git add <filename>
{% endhighlight %}

Git commit 하기
{% highlight ruby %}
 git commit -m "message"
{% endhighlight %}

Git push 하기
{% highlight ruby %}
 git push
{% endhighlight %}

git commit 기록 확인하기
{% highlight ruby %}
git log
{% endhighlight %}

branch 생성
{% highlight ruby %}
git branch <branch_name>
{% endhighlight %}

branch 전환
{% highlight ruby %}
git checkout <branch_name>
{% endhighlight %}

branch 병합
{% highlight ruby %}
git merge <branch_name>
{% endhighlight %}

branch 삭제
{% highlight ruby %}
git branch -d <branch_name>
{% endhighlight %}
